# دليل التعامل مع الخط العربي (Somar) في تطبيق خدوم

تم تحديث الخط العربي الافتراضي للمشروع إلى خط **Somar** بجميع أوزانه. هذا الدليل يشرح كيفية إدارة واستخدام هذا الخط للمبرمجين.

## ١. ملفات الخطوط
تم حفظ ملفات الخطوط في المجلد التالي:
`fonts/`
- `Somar-Thin.otf` (100)
- `Somar-Light.otf` (300)
- `Somar-Regular.otf` (400)
- `Somar-Medium.otf` (500)
- `Somar-Bold.otf` (700)
- `Somar-Black.otf` (900)

## ٢. التكامل مع CSS
تم تعريف الخط باستخدام قواعد `@font-face` في ملف `styles/tokens.css` (وفي صفحات HTML المستقلة). 

يتم الوصول للخط عبر متغيرات CSS (CSS Variables) لضمان سهولة التغيير مستقبلاً:
```css
:root {
  --font-ar: 'Somar', sans-serif;
  --font-display: 'Space Grotesk', 'Somar', sans-serif;
}
```

## ٣. الاستخدام في الكود
بدلاً من كتابة اسم الخط مباشرة، استخدم دائماً المتغيرات:
```css
.my-element {
  font-family: var(--font-ar);
  font-weight: 500; /* سيستخدم تلقائياً Somar-Medium */
}
```

## ٤. ملاحظات هامة للمطور القادم
- **الأوزان المتاحة**: تأكد من استخدام الأوزان (100, 300, 400, 500, 700, 900) المتوفرة في ملفات OTF.
- **تحديث الخط**: إذا رغبت في تغيير الخط العربي مستقبلاً، كل ما عليك فعله هو استبدال الملفات في مجلد `fonts` وتحديث مسارات `@font-face` في `styles/tokens.css`.
- **بدون تبعات خارجية**: تم الاعتماد كلياً على ملفات الخط المحلية لضمان السرعة والاتساق.
