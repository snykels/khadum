1️⃣ قسم إدارة المستقلين
===== القسم: إدارة المستقلين (Freelancers Management) =====

🔍 البحث:
- Dribbble: "user management dashboard", "talent platform admin"
- إلهام من: Upwork admin, Fiverr pro dashboard, LinkedIn Recruiter, 
  Toptal admin, منصة بعيد، مستقل، خمسات

🎯 المميزات المطلوبة (فعّلها كلها):

1. جدول المستقلين الذكي:
   - عرض كـ table + cards view (toggle)
   - أعمدة: الصورة، الاسم، التخصص، التقييم، عدد المشاريع، 
     الحالة، الأرباح، تاريخ التسجيل، آخر نشاط
   - ترتيب (sort) على كل عمود (تصاعدي/تنازلي)
   - ترتيب أبجدي من أ-ي و ي-أ
   - ترتيب بالتقييم الأعلى/الأقل
   - ترتيب بالأحدث/الأقدم تسجيلاً
   - ترتيب بعدد المشاريع المكتملة
   - ترتيب بالأرباح

2. فلاتر متقدمة (Multi-filter):
   - التخصص (dropdown متعدد الاختيار)
   - الحالة: نشط / موقوف / قيد المراجعة / محظور / غير موثق
   - التقييم: من 1 إلى 5 نجوم (range slider)
   - عدد المشاريع: أقل من X / بين X و Y / أكثر من Z
   - تاريخ التسجيل: اليوم / الأسبوع / الشهر / السنة / مخصص
   - الجنسية / المدينة
   - موثّق / غير موثّق (الهوية)
   - متصل الآن / غير متصل
   
3. البحث الفوري:
   - شريط بحث يبحث في: الاسم، الجوال، البريد، رقم الهوية، رقم المستقل
   - Auto-complete مع عرض النتائج مباشرة
   - البحث الضبابي (fuzzy search) — يقبل أخطاء إملائية

4. العمليات المجمّعة (Bulk Actions):
   - اختيار متعدد بـ checkbox
   - تفعيل/تعطيل مجموعة دفعة واحدة
   - تصدير Excel/CSV/PDF
   - إرسال إشعار واتساب جماعي
   - حذف مجموعة (مع تأكيد)

5. بطاقة تفاصيل المستقل (Detail Panel):
   - تفتح كـ slide-over من اليمين (drawer)
   - معلومات شخصية + وثائق الهوية
   - إحصائيات: مشاريع مكتملة، ملغاة، إجمالي الأرباح، متوسط التقييم
   - سجل المشاريع آخر 30 يوم (timeline)
   - سجل المحادثات
   - سجل المدفوعات والسحوبات
   - المراجعات والتقييمات من العملاء
   - زر موافقة/حظر/إيقاف مؤقت مع حقل السبب
   - زر فتح محادثة واتساب مباشرة

6. الإجراءات السريعة (Quick Actions) لكل صف:
   - قائمة (⋮) بـ: عرض التفاصيل، تعديل، موافقة، حظر، 
     إرسال رسالة، عرض المشاريع، عرض المحافظ المالية

7. إحصائيات في الأعلى (KPI Cards):
   - إجمالي المستقلين + النسبة المئوية للزيادة هذا الشهر
   - النشطين الآن
   - قيد المراجعة (بادج أحمر إذا كان العدد مرتفع)
   - متوسط التقييم العام

8. ميزات ذكية إضافية:
   - حفظ الفلاتر المفضلة (Saved Views)
   - مقارنة بين مستقلَين (Compare)
   - تصدير تقرير أداء المستقل PDF
   - أيقونة بجانب كل مستقل تُظهر حالته (🟢 online، 🟡 busy، ⚫ offline)
   - إشعار إذا كان المستقل لم يستجب لآخر رسالة خلال > ساعة

⚡ التفعيل:
- كل فلتر يعدّل URL (querystring) حتى الرابط قابل للمشاركة
- كل زر مربوط بـ API حقيقي من PostgreSQL
- اختصارات: Ctrl+K للبحث، Ctrl+A لاختيار الكل، Del للحذف
- real-time updates (المستقل الجديد يظهر فوراً)
- pagination: 20/50/100 صف لكل صفحة + infinite scroll اختياري

🎨 التصميم:
- استلهم من Linear.app و Notion databases
- استخدم badges ملوّنة للحالات
- أيقونات Lucide
- animations خفيفة عند التحميل

🛡️ لا تحذف أي ميزة موجودة حالياً في القسم، فقط أضف هذه المميزات وحسّن.

2️⃣ قسم الطلبات والمشاريع
===== القسم: الطلبات والمشاريع (Orders & Projects) =====

🔍 البحث:
- Dribbble: "order management dashboard", "project kanban board"
- إلهام من: Trello, Asana, Monday.com, Jira, Linear issues,
  Shopify orders, Stripe payments

🎯 المميزات المطلوبة:

1. عرضان قابلان للتبديل (View Modes):
   - Table view (جدول كلاسيكي)
   - Kanban board (أعمدة حسب الحالة: جديد → قيد التنفيذ → 
     مراجعة → مكتمل → ملغي → نزاع)
   - Calendar view (تقويم حسب تاريخ التسليم)
   - Timeline/Gantt view (للمشاريع الطويلة)

2. فلاتر متقدمة:
   - الحالة (multi-select)
   - نطاق السعر (slider: من - إلى)
   - التصنيف/الخدمة
   - العميل (بحث ذكي)
   - المستقل (بحث ذكي)
   - نطاق تاريخ الطلب (date range picker)
   - نطاق تاريخ التسليم
   - المتأخرة عن موعد التسليم (toggle)
   - التي فيها نزاع (toggle)
   - طريقة الدفع (مدى، Apple Pay، فيزا...)

3. ترتيب (Sort):
   - الأحدث / الأقدم
   - الأعلى / الأقل سعراً
   - الأقرب / الأبعد في التسليم
   - حسب الحالة
   - أبجدياً باسم العميل / المستقل

4. البحث:
   - بحث في: رقم الطلب، اسم العميل، اسم المستقل، 
     وصف المشروع، رقم المعاملة في Tap

5. بطاقة تفاصيل الطلب:
   - معلومات العميل والمستقل (مع صور)
   - وصف المشروع كامل
   - الملفات المرفقة (عرض/تحميل)
   - سجل المحادثة بين الطرفين (timeline)
   - سجل التغييرات (status history)
   - تفاصيل الدفع من Tap Payments
   - زر تدخّل الأدمن (حل نزاع، استرداد، إلغاء)
   - إضافة ملاحظات إدارية داخلية (لا يراها العميل/المستقل)

6. عمليات مجمّعة:
   - تصدير Excel/PDF
   - تغيير حالة مجموعة
   - إرسال تذكير للمستقلين المتأخرين

7. إحصائيات في الأعلى:
   - إجمالي الطلبات اليوم/الأسبوع/الشهر (مع trend ↑↓)
   - قيمة الطلبات الإجمالية
   - متوسط قيمة الطلب
   - معدل الإلغاء
   - معدل التسليم في الوقت
   - عدد النزاعات الحالية

8. ميزات ذكية:
   - تنبيه تلقائي للطلبات المتأخرة > 24 ساعة
   - تنبيه للطلبات التي لم يُرد عليها > 2 ساعة
   - اقتراح مستقل بديل إذا اعتذر الأصلي
   - عرض الهامش الربحي لكل طلب (السعر - عمولة واتساب - عمولة Tap)
   - طباعة فاتورة / إيصال PDF جاهز
   - نسخ رابط الطلب للمشاركة

⚡ كل ميزة مربوطة بـ API حقيقي + PostgreSQL + real-time updates.

🛡️ لا تحذف أي ميزة موجودة.

3️⃣ قسم المحافظ والسحوبات
===== القسم: المحافظ المالية والسحوبات =====

🔍 البحث:
- Dribbble: "wallet dashboard", "payout management", "fintech admin"
- إلهام من: Stripe Dashboard, PayPal admin, Wise business,
  Binance admin, منصة باي بال

🎯 المميزات:

1. جدول المعاملات المالية:
   - أعمدة: رقم المعاملة، النوع (إيداع/سحب/عمولة/استرداد)، 
     المبلغ، المستخدم، الحالة، التاريخ، طريقة الدفع
   - ألوان مميزة: أخضر للإيداع، أحمر للسحب
   - badges للحالات

2. فلاتر:
   - نوع المعاملة
   - نطاق المبلغ (من - إلى)
   - الحالة (مكتملة، قيد المعالجة، فاشلة، مستردة)
   - طريقة الدفع (Tap، تحويل بنكي، محفظة)
   - نطاق التاريخ
   - المستخدم (عميل/مستقل)

3. ترتيب:
   - الأحدث/الأقدم
   - الأعلى/الأقل مبلغاً
   - حسب الحالة

4. بحث:
   - رقم المعاملة، رقم Tap، اسم المستخدم، IBAN

5. إدارة السحوبات (Payouts):
   - قائمة طلبات السحب المعلّقة
   - الموافقة/الرفض بنقرة واحدة مع سبب
   - تأكيد IBAN تلقائياً
   - سحب جماعي (bulk payout)
   - سجل كامل لكل سحب

6. إحصائيات:
   - إجمالي الإيرادات اليوم/الشهر
   - إجمالي العمولات المحصّلة
   - المبالغ قيد الاحتجاز (في انتظار تسليم المشروع)
   - المبالغ المعلّقة للسحب
   - أعلى 10 مستقلين دخلاً

7. ميزات ذكية:
   - كشف حساب لكل مستخدم (PDF/Excel)
   - تقرير ضريبي شهري جاهز
   - تنبيه على المعاملات المشبوهة (أكبر من حد معين، تكرار سريع)
   - تسوية تلقائية مع Tap في نهاية اليوم
   - Reconciliation: مقارنة أرقامنا مع Tap

⚡ مربوط بـ Tap Payments API للاستعلام اللحظي.
