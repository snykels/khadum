## كيف يعمل المنتج
### للعميل
1. يرسل الطلب عبر واتساب.
2. البوت يجمع الوصف والميزانية والمدة.
3. يتم إنشاء طلب حقيقي في جدول `orders`.
4. يبدأ البحث عن المستقلين.
5. يتم إرسال العرض للمستقل واحداً تلو الآخر.
6. عند قبول العرض، يتم إنشاء رابط الدفع تلقائياً.

### للمستقل
1. يصل العرض عبر واتساب.
2. يرد بنعم أو لا.
3. إذا قبل، يتم ربطه بالطلب وتحديث حالة الطلب.
4. بعد الدفع يبدأ التنفيذ.

## ما تم تنفيذه حتى الآن
- تحويل التطبيقات المقبولة إلى حسابات مستقلة فوراً.
- إعادة تصميم صفحة الطلبات والدعوات في لوحة الأدمن.
- شارات جانبية ديناميكية للطلبات والمستقلين.
- قبول / رفض / إعادة إرسال الدعوات من لوحة الأدمن.
- إنشاء رابط التفعيل وإرساله عبر البريد وواتساب.
- تقليل تسريب التوكنات في ردود الأدمن.
- دعم تفعيل الحسابات الحالية بدل إنشاء نسخ مكررة.
- تحسين الردود الخلفية للبوت لتعمل بشكل موثوق.
- إضافة كشف تسريب أرقام التواصل قبل الإرسال للطرف الآخر.
- إنشاء طلبات حقيقية في جدول `orders` عند اكتمال intake.
- إرسال عروض للمستقلين بشكل متسلسل مع مهلة رد دقيقتين.
- ربط ردود المستقلين بالجلسة والطلب الصحيح.
- إنشاء رابط الدفع تلقائياً بعد قبول المستقل.
